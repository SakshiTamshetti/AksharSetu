## 🛑 The Problem

Today, many people are unable to read signboards, directions, or public notices in local languages due to their inability to read a different script. A traveler from Andhra Pradesh may recognize spoken words in Punjab but find themselves lost when faced with a Gurmukhi signboard. This is not a language problem—it is a **script barrier**, where meaning exists but readability is lost.

## 💡 The Solution

AksharSetu focuses on **transliteration rather than just translation**, converting text from one script into another while preserving its original pronunciation and identity. By removing the barrier of scripts while preserving linguistic identity, AksharSetu transforms Bharat’s diversity from a challenge into a strength.

### 💎 Dual-Layer Intelligence
We provide a dual-layer output:
1. **Transliteration:** Instant conversion into the user's familiar script (e.g., Gurmukhi to Telugu script).
2. **Translation:** Optional contextual translation to ensure full comprehension of the meaning.

## ✨ Key Features

### 📸 Real-Time Vision
- **Live Camera:** Point your camera at any signboard to see it transliterated and translated instantly.
- **Smart Glasses Mode:** An immersive, AR-style overlay designed for heads-up displays, creating an augmented, subtitle-like effect.
- **Image Upload:** Process photos of signboards or documents to extract and transliterate text.

### 🗣️ Multimodal Interaction
- **Voice-to-Script:** Speak in one language and see it rendered in your target script.
- **Text-to-Text:** High-precision transliteration for over 20+ Indian and global scripts.
- **Speech Synthesis:** Listen to the correct pronunciation of transliterated text.

### 🗺️ Travel & Culture
- **Travel Guide:** Get real-time safety status, local tips, and "must-know" phrases for your current location.
- **Symbol Library:** A curated collection of cultural, religious, and functional symbols. AksharSetu recognizes common symbols and public signs, converting them into both script and meaning.
- **Cultural Context:** AI-generated insights into the history and usage of specific words or symbols.

### 🎨 Premium Design
- **Alive Background:** An organic, breathing background that shifts and scales gently.
- **Sweet Minimal UI:** A refined, high-end interface that prioritizes clarity and ease of use.
- **Theme Support:** Fully optimized Light and Dark modes with persistent user preferences.

## 🛠️ Getting Started

### Prerequisites
- Node.js (v18 or higher)
- npm or yarn
- A Gemini API Key

### Installation

1. **Clone the repository:**
   ```bash
   git clone https://github.com/your-username/aksharsetu.git
   cd aksharsetu
   ```

2. **Install dependencies:**
   ```bash
   npm install
   ```

3. **Set up environment variables:**
   Create a `.env` file in the root directory and add your Gemini API key:
   ```env
   GEMINI_API_KEY=your_api_key_here
   ```

4. **Start the development server:**
   ```bash
   npm run dev
   ```

5. **Build for production:**
   ```bash
   npm run build
   ```

## 📖 Usage

- **Camera Mode:** Grant camera permissions and point at text. Use the "Capture" button for a deep analysis.
- **Travel Guide:** Enter a location to get safety ratings and cultural tips.
- **Symbol Library:** Click the "Symbols" icon in the header to browse or search for cultural icons.
- **Smart Glasses:** Toggle the "AR Vision" mode for a high-contrast, heads-up display interface.
